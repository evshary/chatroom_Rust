# chatroom_Rust

A chatroom example written in Rust.

# Usage

* Normal server & client

```bash
# Run server
cargo run --bin server <port>
# Run client
cargo run --bin client <IP> <port>
```

* Async server & client

```bash
# Run server
cargo run --bin async_server <port>
# Run client
cargo run --bin async_client <IP> <port>
```
